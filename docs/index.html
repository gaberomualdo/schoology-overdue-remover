Schoology Overdue Remover

Removes the overdue box on Schoology (ASL)!

    Download (.zip)

Download Instructions

1. Click download button and get the Schoology-Overdue-Remover.zip file.

2. Unzip the Schoology-Overdue-Remover.zip file — it should be in your downloads folder.

3. Move the Schoology-Overdue-Remover folder (located in the Downloads folder) to a directory where you will likely not move it or delete it — e.g. the Documents folder.

4. Open Google Chrome (or Chromium, Brave, or another Chromium-based browser) and click on the menu (3 dots stacked) icon on the top right corner, then click on More Tools > Extensions.

5. Turn on Developer Mode on the top right corner by sliding the slider right, or clicking on the slider.

6. Go to the top left hand corner and click the Load Unpacked button, then open the unzipped Schoology-Overdue-Remover folder.

Note (License, Releases, Usage, etc.)

Schoology Overdue Remover is not a public Chrome extension, however is MIT Licensed, meaning anyone can distribute it freely given they include the LICENSE.txt file included.

Schoology Overdue Remover only works on lms.asl.org on Chromium-based web browsers with extension capabilities.

Ports of this software may be made available in the future on other browsers such as Firefox, Opera, and more, in future releases. It is only available on Desktop devices.

The latest release was on February 21, 2020. More information can be found on the README.md file in the zipped source.
